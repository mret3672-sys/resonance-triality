## Triality Scaffolding

The generative grammar. Any domain gets decomposed this way.

```
1 Entity        = 1 Node
2 Entities      = 1 Duality
Parent + Duality = 1 Triality
3 Trialities    = 1 Tile (Cell)
9 Cells         = 1 Table
```

The scaffolding is fractal -- a Table of Tables is just the next level up.
